# DB-Merchandising-ProjetoEP4
Repositório final do projeto acadêmico de Modelagem de Banco de Dados. Contém scripts SQL (CREATE/INSERT/SELECT) para PostgreSQL de um sistema de **Gestão e Controle de Materiais de Merchandising**. O modelo lógico está em **3ª Forma Normal (3FN)**, garantindo integridade de dados e rastreabilidade completa de estoque, clientes e colaboradores.
